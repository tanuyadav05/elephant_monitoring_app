# Elephant Monitor App

## Overview
This is a React Native application for monitoring elephants in real-time.

## Getting Started
1. Clone the repository.
2. Navigate into the directory: cd ElephantMonitorApp.
3. Install dependencies: npm install.
4. Start the app: npm start or npx expo start.

## Features
- View elephants on a map
- Alerts page to show real-time updates
- Basic settings functionality

## Technologies Used
- React Native
- React Navigation
- Context API
